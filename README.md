# ✈️ SkyWay - Modern Flight Booking App

A modern, scalable flight booking web application built with Next.js, enabling users to search, compare, and book flights seamlessly.

SkyWay is designed for speed, flexibility, and a premium user experience. It leverages the latest features of **Next.js 16**, **React 19**, and **Tailwind CSS 4** to provide a lightning-fast interface for global flight discovery.

---

## ✨ Features

- **🚀 Smart Flight Search**: Blazing fast search with real-time updates.
- **🔍 Advanced Filtering**: Filter by price, duration, airlines, and layovers.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **⚡ Performance First**: Zero-latency transitions and optimized asset loading.
- **🔒 Secure Booking**: A robust, multi-step checkout process.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **State Management**: React Server Components & Hooks
- **Icons**: Lucide React
